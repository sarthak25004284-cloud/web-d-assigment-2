Student Portfolio Website

This is a multi-page student portfolio website created using only HTML and CSS.
It includes four pages: Home, About, Skills, and Contact.
A common navigation bar is used across all pages for smooth navigation.
The About page contains a profile image and a short introduction.
The Skills page displays multiple project/skill cards using Flexbox layout.
The Contact page includes a styled contact form with Name, Email, Subject, and Message fields.
The website demonstrates proper HTML structure, CSS styling, Flexbox usage, and clean folder organization.
